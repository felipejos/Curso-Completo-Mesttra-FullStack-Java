# Complemento da Lição

## 1) Como estudar sem travar (método de 3 passos)
Para cada exercício do livro, faça sempre nesta ordem:

1. **Entradas**: o que o usuário vai digitar?
2. **Processamento**: quais contas/decisões você precisa fazer?
3. **Saídas**: o que precisa aparecer no final?

---

## 2) Conversão rápida (Português Estruturado → Java)
Quando o enunciado (ou o livro) tiver algo como:

- **Leia** X → `Scanner` + `nextLine()` / `nextInt()` / `nextDouble()`
- **Escreva** X → `System.out.println(...)`
- **Se / Senão** → `if (...) { } else { }`
- **Enquanto** → `while (...) { }`
- **Para** → `for (...) { }`

---

## 3) Mini-checklist de qualidade (antes de seguir para o próximo exercício)
- [ ] Dei nomes claros para variáveis (`salario`, `idade`, `area`, `total`)
- [ ] Usei `double` quando envolve dinheiro/decimal
- [ ] Imprimi mensagens com rótulo (ex.: `Área:`, `Total:`)
- [ ] Testei com um valor pequeno e um valor maior

---
