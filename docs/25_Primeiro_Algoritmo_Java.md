# Complemento da Lição

## 1) O que “algoritmo” significa (bem simples)
**Algoritmo** é uma **receita de passos** para resolver um problema.

### Exemplo do mundo real
Receita de miojo:
1) ferver água  
2) colocar macarrão  
3) esperar 3 minutos  
4) colocar tempero  

Em programação é igual: passos claros, na ordem certa.

---

## 2) Checklist de prática (faça junto com o vídeo)
- [ ] Criei a classe `Main` (ou `App`)
- [ ] Usei `public static void main(String[] args)`
- [ ] Imprimi algo com `System.out.println`
- [ ] Pedi um dado para o usuário com `System.out.print`
- [ ] Li um dado usando `Scanner`
- [ ] Mostrei o resultado final para o usuário

---

## 3) Mini-atividade (pra fixar sem complicar)
Faça um algoritmo com 3 passos:
1) pedir o nome
2) pedir a idade
3) imprimir: `Olá <nome>, você tem <idade> anos.`
