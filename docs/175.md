# Complemento da Lição

- Para **reproduzir o vídeo sem sair da página**, o renderizador do Markdown precisa aceitar HTML com `<iframe>`.
- Em **README do GitHub**, normalmente o `<iframe>` **não é renderizado**, então o vídeo **não toca embutido**.
- Nesse caso, a melhor apresentação visual é:
  - manter o link do vídeo;
  - usar a imagem clicável;
  - ou hospedar o vídeo diretamente como arquivo suportado pelo GitHub, quando aplicável.
- O ID do vídeo usado para a incorporação foi: `ZtMzB5CoekE`.
- O formato `https://www.youtube.com/embed/ZtMzB5CoekE` é o modelo correto para tentativa de exibição embutida.
