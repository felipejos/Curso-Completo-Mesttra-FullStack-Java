## 3) Conceitos (simples + exemplo do mundo real)

### IP
“Endereço” de uma máquina na rede.  
Exemplo: como o endereço de uma casa.

### Porta
“Porta de entrada” de um serviço dentro da máquina.  
Exemplo: apartamento dentro do prédio (mesmo endereço, portas diferentes).

### Web Server
Programa que recebe requisições HTTP e responde.  
Exemplo: atendente que recebe pedidos e devolve respostas.

### HTTP (Request/Response)
Um “pedido” (request) e uma “resposta” (response).  
Exemplo: você pede um lanche e recebe o lanche + comprovante.

### GET vs POST
- **GET**: buscar dados (sem alterar).  
- **POST**: enviar dados (normalmente cria algo).  
Exemplo: GET = olhar o cardápio, POST = fazer o pedido.

### Status Code
Código que diz o resultado. Exemplos comuns:
- **200**: ok  
- **201**: criado  
- **400**: erro no pedido  
- **404**: não encontrado  
- **500**: erro no servidor
