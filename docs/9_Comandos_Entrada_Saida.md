# Complemento da Lição

## 1) O que é “entrada e saída” (bem simples)

- **Entrada (input)**: quando o programa **recebe** dados.
  - Ex.: você digita seu nome no teclado.
- **Saída (output)**: quando o programa **mostra** algo.
  - Ex.: o programa imprime “Olá, Felipe”.

📌 Em programas de **console**, a entrada geralmente vem do **teclado** e a saída vai para a **tela**.

---

## 2) “Programa em terminal de console” (explicação + exemplo do mundo real)

### O que é
É um programa que roda no terminal (Prompt/PowerShell/Bash) e conversa com o usuário por texto.

### Exemplo do mundo real
Pensa num atendimento por mensagem:
- você envia uma mensagem (entrada)
- a pessoa responde (saída)

Só que aqui é o programa fazendo isso no terminal.

---

## 3) Quando faz sentido usar console (casos comuns)

- Aprender lógica e programação (mais simples, sem tela)
- Scripts e automações (renomear arquivos, gerar relatórios, etc.)
- Ferramentas internas (processar dados rapidamente)
- Backends e servidores (muitas vezes rodam sem interface gráfica)
- Testar ideias rápido (protótipos)
