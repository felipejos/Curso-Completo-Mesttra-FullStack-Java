# Complemento da Lição

## 1) Objetivo do vídeo (o que você deve sair sabendo)
Ao finalizar, você deve conseguir:
- identificar **+ - * / %** no código sem travar
- entender a diferença entre **divisão inteira** e **divisão com decimal**
- usar `%` para descobrir **par/ímpar** e “resto” de divisão

---

## 2) Checklist de prática (faça junto com o vídeo)
- [ ] Testei `+` com números (soma)
- [ ] Testei `+` com texto (concatenação)
- [ ] Testei divisão com `int` (corta decimal)
- [ ] Testei divisão com `float/double` (mantém decimal)
- [ ] Testei `%` para resto e par/ímpar

---

## 3) Exercícios rápidos (fixação)

1) Sem executar, tente prever:
    System.out.println(8 / 3);
    System.out.println(8 % 3);

2) Faça sair `2.666...` em vez de `2` usando 8 e 3.

3) Escreva um programa que leia um número e imprima:
- “PAR” se `n % 2 == 0`
- “ÍMPAR” caso contrário

---

## 4) Pergunta única (para checar que fixou)
Quando você faz `10 / 6` usando `int`, o Java mantém as casas decimais ou corta?
