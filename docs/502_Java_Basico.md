## Complemento da Lição

### 🎯 Para que serve esse roadmap
Usar como **trilha** para estudar Java do zero até ter base suficiente para:
- escrever programas no console com segurança (entrada/saída, cálculos, condições, repetição)
- entender **POO** (classes, objetos, métodos, herança, polimorfismo)
- chegar preparado para **Spring Boot** (controllers, services, APIs)

---

### 🧭 Ordem prática de estudo (sequência recomendada)
1) **Sintaxe básica**
   - variáveis, tipos, operadores, `System.out.println`
2) **Decisão e repetição**
   - `if/else`, `switch`, operador ternário
   - `for`, `while`, `do-while`
3) **Estruturas básicas**
   - `String`, arrays, métodos/funções
4) **POO (o coração do Java)**
   - classes/objetos, atributos, métodos
   - construtor, encapsulamento (`private` + getters/setters)
   - herança, polimorfismo, `abstract`, `interface`
5) **Coleções (quando já estiver confortável)**
   - `List`, `Set`, `Map` (uso básico)
6) **Erros e robustez**
   - `try/catch`, validação de entrada, mensagens claras

---

### ✅ Como estudar sem “passar batido”
Para cada tópico:
- Leia o conceito
- Faça 3 exercícios:
  - 1 fácil (entender a sintaxe)
  - 1 médio (com regra de negócio)
  - 1 com validação (`try/catch` + repetição)

---

### 🧩 Mini-desafios (bem rápidos) para cada bloco
- **Sintaxe/Tipos**: ler 2 números e imprimir soma, diferença, produto e divisão
- **If/Else**: média do aluno (aprovado/recuperação/reprovado)
- **Switch**: dia da semana (1..7 + inválido)
- **Loops**: repetir até o usuário digitar “sair”
- **Métodos**: criar `calcularIMC(peso, altura)` e `classificarIMC(imc)`
- **POO**: classe `Aluno` com notas e método `media()`
- **Try/Catch**: impedir que o programa quebre quando digitar letra no lugar de número

---

### 🔗 Conexão direta com o seu “Primeiro Projeto Spring Boot”
O Spring Boot vai ficar muito mais fácil quando você já dominar:
- `if/else` e `switch` (regras de negócio)
- métodos (organizar código)
- POO (modelos/entidades/DTOs)
- listas/mapas (retornar coleções em endpoints)
