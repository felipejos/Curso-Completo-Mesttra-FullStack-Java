# Introdução ao Banco de Dados

## 🎯 Objetivo
Criar uma base sólida para entender:
- o que é **Banco de Dados**;
- por que ele existe e **quando usar**;
- principais conceitos iniciais para você acompanhar as próximas aulas com confiança.
