# Complemento da Lição

## 1) O que é uma linguagem de programação (sem jargão)
Uma **linguagem de programação** é uma forma “padronizada” de escrever instruções para o computador executar.

### Exemplo do mundo real
- Português/Inglês: você fala com pessoas.
- Linguagem de programação: você “fala” com o computador.

---

## 2) Como escolher uma linguagem (regra simples)
A melhor linguagem é a que combina com:
- **o tipo de trabalho** que você quer (web, mobile, dados, jogos)
- **o ecossistema** (bibliotecas, ferramentas, mercado)
- **o que você já está estudando** (consistência)

No seu contexto atual (Java + MySQL + API), faz sentido continuar firme em **Java + Backend** até dominar bem.
