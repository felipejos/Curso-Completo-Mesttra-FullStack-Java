## 🚀 Como utilizar
1. Faça o clone ou download do repositório.
2. Navegue até a pasta `docs/` e abra o material desejado.
3. Marque seus progressos e anote dúvidas para compartilhar nas mentorias.

## 🧰 Pré-requisitos sugeridos
- Navegador atualizado para acessar os vídeos.
- Editor de código (VS Code, IntelliJ ou similar) para praticar.
- JDK 17+ instalado para acompanhar aulas práticas de Java.

## 🤝 Como contribuir
1. Abra uma issue descrevendo a sugestão ou correção.
2. Faça um fork, implemente melhorias e abra um Pull Request.
3. Informe claramente qual material foi alterado ou adicionado.

## 📬 Suporte
Se identificar algum link quebrado ou quiser sugerir novos recursos, registre uma issue ou entre em contato com a coordenação do curso.
